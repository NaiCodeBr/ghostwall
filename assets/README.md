# Assets

Este diretório contém os assets visuais do projeto GHOSTWALL.

## Logo

Coloque a logo do projeto em `assets/logo.png`.

### Especificações

- **Formato:** PNG
- **Dimensões:** 1200x400 pixels (recomendado)
- **Tamanho máximo:** 500KB
- **Fundo:** Escuro (preferencialmente #1a1a2e)
- **Texto:** "GHOSTWALL RF Privacy Firewall"

### Como usar

1. Coloque sua imagem como `assets/logo.png`
2. O arquivo está no `.gitignore`, então não será commitado
3. O README.md está configurado para exibir `assets/logo.png`
4. Se a imagem não existir, o GitHub mostrará um link quebrado

### Nota

Se você quiser incluir a logo no repositório, remova `assets/logo.png` do `.gitignore`.

## Contribuidores

Este projeto agradece a todos os contribuidores que ajudaram a tornar o GHOSTWALL possível.

## Como Ser Reconhecido

Contribua com o projeto e seu nome será adicionado a esta lista:

- Reportar bugs
- Sugerir features
- Submeter pull requests
- Melhorar a documentação
- Traduzir para outros idiomas
- Divulgar o projeto
